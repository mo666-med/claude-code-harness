# /project-overview

以下を短くまとめてください（箇条書き中心）。

## 入力

- @README.md
- @Plans.md
- @AGENTS.md
- @CLAUDE.md

可能なら：
- `git status -sb`
- `git log --oneline -10`

## 出力フォーマット

- **目的**: このプロジェクトが何を作っているか（1-2行）
- **現在地**: 今のフェーズ/未完了タスク（Plans.md基準）
- **技術**: 言語/主要フレームワーク/テスト手段（推測でOK）
- **次アクション**: 3つまで
